# Core Asset Manager - Zoho Creator Widget

A custom Zoho Creator Widget built to display and manage company assets in a visually appealing dashboard.

## 📌 What it does
- Displays asset categories as interactive cards on the Home dashboard
- Categories include: **Laptop, Mouse, Headphone, License, Other**
- On clicking any card, the specific asset details open automatically
- Clean and responsive UI built inside Zoho Creator

## 🛠️ Tech Stack
- **Zoho Creator** — Platform
- **Zoho Widget SDK** — Widget framework
- **JavaScript** — Logic and interactions
- **HTML/CSS** — UI and styling
- **Node JS** — Local development server
